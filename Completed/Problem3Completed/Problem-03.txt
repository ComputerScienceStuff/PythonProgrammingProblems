03 - Double Number List

Input:
The input consists of a two integers per line, separated by a comma. 
Some integers are less than zero.

Output:
Write a program to check if the two numbers add up to a sum greater than 5.  
If the sum is greater than 5, output a 1.  
If not greater than 5, output a 0.  
Write each digit on a separate line.

Write your answers in a file named 03-TeamName.txt, one answer per line.



Example:
0,2
1,3
2,2
3,2
4,1
5,2
6,2
7,2
8,2
9,2
-10,2
100,2
1000,2
