02 - Single Number List

Input:
The input consists of a single integer per line.

Output:
Write a program to read in each line of the input file. 
Check the if the input number is greater than 5.  
If the number is greater than 5, output a 1.  
If not greater than 5, output a 0.  
Write each digit on a separate line.

Write your answers in a file named 02-TeamName.txt, one answer per line.
