04 - Leading Capitalization

Input:
The input consists of a word from the dictionary.  
Random letters in the word are capitalized.

Output:
Make sure the first letter in the word is capitalized.  
Other than the first letter, all remaining letters must retain their original case.  That is, lowercase stays lower and uppercase stays upper.

Write your answers in a file named 04-TeamName.txt, one answer per line.


Example:
Run
spoT
run
sEE
jAne
rUn

Answer:
Run
SpoT
Run
SEE
JAne
RUn
